Q1: What does HTML stand for?
Answer: HTML stands for HyperText Markup Language.
Q2: What is HTML used for?
Answer: HTML is used to create and structure the content of web pages. It defines the elements of a webpage, such as headings, paragraphs, links, images, and more.
Q3: What are HTML elements?
Answer: HTML elements are the building blocks of an HTML document. They are made up of: Tags: Indicate the start and end of an element (e.g., <p>and </p>). Content: The information or data placed between the opening and closing tags.
